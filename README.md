# Restaurant Management System (PyQt6)

نظام إدارة مطاعم يعمل بدون إنترنت، بواجهة عربية (RTL)، مبني بـ **Python + PyQt6 + SQLite**.

## الميزات الحالية

- تسجيل دخول الموظفين (كاشير / مدير / صاحب)
- شاشة نقطة بيع (POS) وإنشاء الطلبات
- إدارة المخزون وإضافة الكميات
- إدارة المنيو (الأقسام والأصناف)
- إدارة الموظفين
- الشيفتات (فتح/إغلاق)
- تقارير يومية أساسية
- إعدادات المطعم (ملف إعدادات مشفر)

## المتطلبات

- Python 3.11+ (يفضل)
- نظام Windows (المشروع موجه لتطبيق سطح مكتب)

## التثبيت

```bash
pip install -r requirements.txt
```

## الإعداد الأولي

1. إعداد ملف المطعم المشفر:

```bash
python setup_restaurant.py
```

2. تشغيل التطبيق:

```bash
python main.py
```

عند أول تشغيل سيتم إنشاء قاعدة البيانات والجداول تلقائيًا.

## بيانات الدخول الافتراضية

- **Username:** `admin`
- **Password:** `admin`

> ينشئها التطبيق تلقائيًا إذا لم يوجد أي موظف في قاعدة البيانات.

## هيكل المشروع (مختصر)

```text
database/   # قاعدة البيانات وإنشاؤها
modules/    # منطق الأعمال (auth, orders, inventory, ...)
ui/         # الشاشات (login, pos, inventory, reports, ...)
utils/      # config والتنسيقات والمساعدات
main.py     # نقطة تشغيل التطبيق
```

## ملاحظات

- ملف الإعدادات المشفر: `config.encrypted`
- مفتاح فك التشفير: `config.key`
- إيصالات نصية: `receipts/`
- يوجد مجلد `venv/` داخل المستودع حاليًا (يفضل استبعاده من Git مستقبلًا)

## الاختبارات

لا يوجد حاليًا إطار اختبارات مهيأ داخل المشروع (pytest غير مثبت في البيئة الحالية).
